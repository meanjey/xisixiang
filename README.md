# 习思想刷题系统

一个可直接部署到网页的静态刷题系统，已录入导论和第 1-16 章练习题，共 170 道单选题。

## 页面

- `index.html`：首页与学习概览
- `chapters.html`：章节选择
- `practice.html`：刷题页面
- `review.html`：错题与收藏
- `stats.html`：学习统计

## 功能

- 章节练习、随机练习、错题本、收藏题、模拟考试
- 自动判分、答案解析、答题卡跳转
- 本地保存练习记录、错题和收藏
- 手机端适配，可作为 GitHub Pages 静态站点部署

## 使用

本地直接打开 `index.html`，或用任意静态服务器托管整个目录。

部署到 GitHub Pages 时，站点入口为 `index.html`。
